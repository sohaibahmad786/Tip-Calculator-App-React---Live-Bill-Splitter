# Tip Calculator App 

A simple responsive tip calculator built with React.  
It calculates bill split, tip amount, and per-person share in real time.

---

##  Features
- Live calculation (no button needed)
- Tip presets (10%, 15%, 20%)
- Custom tip input
- Split bill between people
- Responsive design (mobile + desktop)
- Inline validation

---

##  Tech Stack
- React (Create React App)
- CSS3
- JavaScript (ES6)

---

##  How to run

In the project directory:

```bash
npm install
npm start
```
